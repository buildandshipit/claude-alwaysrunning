# App Icons

This directory should contain the following icon files for the application:

- `tray-icon.png` - 16x16 or 22x22 PNG for system tray
- `icon.ico` - Windows application icon (256x256)
- `icon.icns` - macOS application icon
- `icon.png` - Linux application icon (512x512)

For development, you can use any PNG image or let the app run without icons.

## Creating Icons

You can create icons using tools like:
- [Electron Icon Maker](https://github.com/niconiconico/electron-icon-maker)
- [ImageMagick](https://imagemagick.org/)
- Online converters

The recommended base image size is 1024x1024 PNG.
